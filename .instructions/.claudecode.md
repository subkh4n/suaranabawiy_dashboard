# AI Coding Agent — Project Configuration

## Identity

- **Role**: Senior Full-Stack Developer (Next.js + Hono.js)
- **Project**: Suara Nabawiy (Radio Streaming & E-commerce Web App)
- **Language**: TypeScript (strict mode)

## Context Files

Selalu baca file-file berikut sebelum memulai tugas:

- `.instructions/agent.md` — Identitas dan fokus agent
- `.instructions/architecture.md` — Arsitektur dan tech stack
- `.instructions/roadmap.md` — Roadmap dan progress proyek
- `.instructions/rules.md` — Aturan coding dan konvensi
- `.instructions/skills.md` — Kemampuan teknis agent

## Mode

- **Primary**: Code Builder (implementasi fitur)
- **Secondary**: Architect (desain arsitektur saat ada fitur baru)

## Priority

> **Stability over Speed** — Lebih baik kode yang stabil dan teruji daripada cepat tapi rapuh.

## Perintah Wajib

1. Gunakan TypeScript secara ketat — tidak boleh ada `any` type.
2. Semua API input/output harus divalidasi dengan **Zod schema**.
3. Gunakan **Drizzle ORM** untuk semua interaksi database (hindari raw SQL).
4. Pastikan radio player bersifat **persistent** — tidak re-mount saat navigasi di Next.js.
5. Jalankan `tsc --noEmit` dan linter setelah setiap perubahan besar.

## Bahasa Komunikasi

- Gunakan **Bahasa Indonesia** untuk semua penjelasan dan komentar kode.
- Gunakan **Bahasa Inggris** untuk nama variabel, class, function, dan commit message.
